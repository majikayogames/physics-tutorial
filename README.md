# 2D Physics Engine Tutorial - Interactive Guide

An interactive, web-based tutorial for learning how to build a 2D physics engine from scratch. This project accompanies the [2D Physics Engine YouTube Tutorial Series by Majikayo Games](https://www.youtube.com/playlist?list=PLbuK0gG93AsENAa67XysaOr5K0cczxye_) and provides hands-on examples and visualizations for each physics concept.
